## 1. 함수

### 함수의 형태
함수의 구성은 아래와 같다.
```
{반환 값의 자료 형} {함수 이름} ({인자 리스트})
{
    {함수 내용}
    return {반환 값};   // 반환 값이 없으면(void) return 생략가능
}
```
가장 기본적인 함수 구성의 예
```
void function()
{
}
```
